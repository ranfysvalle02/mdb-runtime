PARALLAX

How Parallax Works

Multi-Lens Analysis: Each call is analyzed through SALES, MARKETING, and PRODUCT perspectives using concurrent LLM calls with dynamic schemas.
RAG Snippets: Vector search finds relevant transcript excerpts, then LLM explains why each snippet matters. Powered by mdb-engine's embedding service.
Per-Call Memory: Extracts and stores memories from each call using mdb-engine's Memory component with semantic search.
